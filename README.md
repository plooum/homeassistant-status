HomeAssistant-Status

Le but de ce projet est de créer un service joingnable depuis homeassistant afin de connaître l'état d'un PC et de l'éteidre à distance.
Le service lancera un serveur web https.
Une commande de status répondra l'état on.
Une commande d'extinction prendra en paramètre un mot de passe configurable dans un fichier de configuration.
